# LCM_i2i_PoC
LCM-LoRAを用いた高速i2iの検証用リポジトリ

# 使い方
（Windows環境での実行を想定）

以下のライブラリが使用できるようにセットアップ
- cv2
- numpy
- pygetwindow
- pyautogui
- diffusers
- pyTorch
- PIL
- pynput
- tkinter

PowerShellでrealtime_i2i.pyを実行

生成結果の表示画面が表示されたら、i2iしたい画面をクリック。
